## 搬瓦工换机房后 IP 到底怎么变？三种情况说清楚

很多人对这个问题有误解，以为换了机房就一定是新 IP。实际没那么简单。

**情况一：从机房 A 切到机房 B**

换到一个新机房，IP 肯定会变。这是正常逻辑，不同机房的 IP 池完全不同，不可能给你同一个地址。

**情况二：从机房 B 切回机房 A**

这时候 IP 大概率还是当初在机房 A 的那个，前提是这段时间内没人占用你的旧地址。不过这不是百分之百保证——有时候过了一周回去就换了，有时候过了好几个月再回来还是原来那个。属于随机事件，没有规律。

**情况三：IP 被封后换机房**

这是很多人踩坑的地方。IP 一旦被封，搬瓦工后台会直接禁止你使用机房迁移功能，操作时会报错提示"Migration backend is currently not available for this VPS"。这个时候换机房的路已经堵死了，只剩付费换 IP 一条路。

所以换机房能不能换到新 IP，先要搞清楚你目前的 IP 是什么状态。

## 第一步：确认 IP 状态，别一上来就换

IP 连不上，可能是被封了，也可能只是网络抖动或者本地 DNS 的问题。搬瓦工 KiwiVM 后台自带检测工具，用这个结果做判断最准。

**检测步骤：**

1. 登录搬瓦工客户中心，进入 Services → My Services
2. 找到你的 VPS，点击 KiwiVM Control Panel 进入后台
3. 在浏览器地址栏直接输入检测地址（注意要先登录 KiwiVM，否则会提示会话超时）
4. 点击 **Test Main IP** 开始检测
5. 显示 **IP NOT BLOCKED** 说明 IP 正常，问题出在别处；显示 **IP BLOCKED** 才说明真的被封了

检测结果很快。IP 正常的话，去查查本地网络、DNS 或者 SSH 端口问题；IP 确实被封了，再走后面的流程。

## 第二步：IP 正常——通过换机房换一个新 IP

IP 没被封，但你就是想换一个新地址（比如需要多个 IP 跑不同项目），换机房是最省钱的办法。免费、没有次数限制，随时都能用。

**操作步骤：**

1. 登录 KiwiVM 控制面板
2. 在左侧菜单找到 **Migrate** 分区，点击 **Migrate to another DC**
3. 页面会显示当前机房（标注 current）以及可以迁往的其他机房列表
4. 选择目标机房，点击 **Confirm on next step**
5. 确认后点击 **Start Migration** 开始迁移

迁移完成后会有邮件通知，整个过程通常 15 分钟以内，数据和程序全部保留，不用重新配置。迁过去就是新机房的新 IP。

换机房不收费，但每次迁移会消耗一定流量（跟 VPS 上的数据量有关）。这点要注意，流量不多的套餐别频繁来回折腾。

## 第三步：IP 被封——付费换 IP 操作流程

IP 真的被封了，只能走付费通道。流程也不复杂：

1. 登录搬瓦工客户中心
2. 找到 Services 下的 IP 更换页面
3. 找到需要换 IP 的 VPS，点击对应的 **Request IP Change**
4. 系统自动生成工单，并在账户 Home 页生成一笔待付款订单
5. 完成支付，等待新 IP 下发

付费金额以申请页面显示的实时价格为准（目前大约 8 美元左右，价格可能浮动）。一般情况下付款后很快就会有新 IP 分配下来，偶尔需要人工处理的话最长不超过 24 小时。

新 IP 保证全球连通，不在任何黑名单中。原来的旧 IP 付款后就会被回收，IP 换了之后不需要重新搭建网站或更新服务器配置，数据全部不变。

## 换机房换 IP 的几个常见误区

说实话，这块坑最多的不是操作步骤，而是认知。

**误区一："换机房回来 IP 一定会变"**

实际上大概率不变，原 IP 大概率还在等你。只有旧 IP 被其他用户占用了才会分配新地址。

**误区二："IP 连不上就是被封了"**

不一定。本地网络问题、端口封禁、DNS 污染都会导致类似症状。先用 KiwiVM 的检测工具确认，别直接就付钱。

**误区三："换机房可以绕过 IP 封禁"**

不行。IP 一旦被封，搬瓦工系统会锁定机房迁移功能，这条路走不通。

## 常见问题

**Q：搬瓦工换机房要多久？**

一般 15 分钟以内，最多不超过几个小时，取决于 VPS 上的数据量。迁移完成后会收到邮件通知。

**Q：换机房换 IP 之后，我在 VPS 上搭的东西还在吗？**

全部保留，不需要重新配置。换机房只是换了一个 IP 地址和物理位置，数据不动。

**Q：搬瓦工换机房有次数限制吗？**

没有次数限制，但每次迁移会消耗流量，大套餐没太大影响，小套餐要注意别频繁切换。

**Q：IP 被封之后多久会自动解封？**

搬瓦工没有自动解封机制，被封就是长期封。不要抱着"等一段时间就好了"的念头，该付费换就换，或者换机房换地址。

**Q：FreedomPlan 以外的套餐还有免费换 IP 的方式吗？**

没有。搬瓦工普通套餐的免费换 IP 功能已经下线，不要被老教程误导。
